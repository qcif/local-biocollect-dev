Custom link landing page

This is a landing page for custom links that can be set in Biocollect and other apps.

Search the 'local-biocollect-dev' repository for 'custom_link.html' to find settings that point here.

This page is used as a landing page for services that are not set up in the Local Development environment. This could be because the service is not part of the apps needed for Biocollect, or because it is a service only needed in production.
